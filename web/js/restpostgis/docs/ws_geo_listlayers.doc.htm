List Layers

Service URL

Description

List the geo layers in PostGIS.

Service Arguments

PARAMETER DESCRIPTION
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
layer_name The name of the geotable.
srid The SRID of the layer.
