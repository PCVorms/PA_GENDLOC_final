Metadata Search and Retrieval

Service URL

Description

Searches for and retrieves metadata.

Service Arguments

PARAMETER DESCRIPTION
search Either the search term or the explicit name of the metadata file to retrieve.
field The metadata field in which to search for the search parameter. If left blank, a full text search will take place. This value should be left black if retrieving a specific metadata file.
format Format of the return. For search, format values are "xml" or "json". To retrieve metadata, format options are "raw" (FGDC XML), "classic" (FGDC Classic format), "faq" (FGDC FAQ format), or "summary" (brief metadata summary).

Service Output

Service output for metadata retrieval is dependent upon the metadata layer. Service output for metadata search is as follows:

PARAMETER DESCRIPTION
title The metadata title field content.
abstract The metadata abstract field content.
file_name The metadata file name. Can be used to retrieve specific metadata.
