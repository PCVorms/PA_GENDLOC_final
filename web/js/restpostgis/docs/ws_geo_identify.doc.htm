Identify

Service URL

Description

Identify features from multiple layers. This is a good work-around for when you are trying to identify features from many layers using consecutive Buffer Point requests.

Service Arguments

PARAMETER DESCRIPTION
x The X coordinate of the point to buffer.
y The Y coordinate of the point to buffer.
srid The SRID of the input point. 2264 is NC State Plane NAD83 (feet), 4326 is decimal degrees.
geotables Comma separated PostGIS Layers (address_points,roads,subdivisions)
fields Comma separated fields to be returned separated by layer with "|" (gid,display|gid+road_name|gid,name)
distance The number of units to be buffered. The units are the units of the geotable SRID, and for Mecklenburg County data will be feet.
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

Service output is entirely dependant upon the fields parameter.
