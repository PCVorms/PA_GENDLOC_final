Street Name Search

Service URL

Description

Performs a fuzzy search for a street name in the master address table. Useful for auto complete boxes.

Service Arguments

PARAMETER DESCRIPTION
streetname Street name.
format Format for the output. Value is either 'xml' for xml output, 'json' for JSON output, and 'text' for plain text output.

Service Output

PARAMETER DESCRIPTION
street_name Street name.
