Centerline Intersection

Service URL

Description

Returns the intersection(s) of two streets if one exists.

Service Arguments

PARAMETER DESCRIPTION
streetname1 First street name to search for.
streetname2 Second street name to search for.
srid The SRID (projection) to return the intersection points in. 2264 is NC State Plane NAD83 Feet; 4326 is WGS84.
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
xcoord X coordinate of intersection.
ycoord Y coordinate of intersection.
