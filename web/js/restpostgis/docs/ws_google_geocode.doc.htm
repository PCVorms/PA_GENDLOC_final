Google Geocode

Service URL

Description

Uses Google's geocoding service to geolocate an address.

Service Arguments

PARAMETER DESCRIPTION
address Address to geocode. Example: 5501 Ruth Dr., Charlotte, NC
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

Output format is that specified by the Google Maps API. For more details, click here.
