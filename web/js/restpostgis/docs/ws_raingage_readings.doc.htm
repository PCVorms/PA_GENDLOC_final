Rain Gage Readings

Service URL

Description

Retrieve the readings of all the Rain Gages inside Mecklenburg County during a certain time frame (ex: last 60 minutes).

Service Arguments

PARAMETER DESCRIPTION
timeframe Timeframe (in minutes), for which rainfall readings are requested
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
sensorid Rain Gage Number used by Charlotte Mecklenburg Storm Water Services (Alert).
usgs_station_no Rain Gage Number used by USGS (NWIS).
sitename Rain Gage Name.
longitude Longitude.
latitude Latitude.
rainamt Amount of rainfall recorded during the requested timeframe in inches.
