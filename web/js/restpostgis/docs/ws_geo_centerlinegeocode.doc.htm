Centerline Geocode

Service URL

Description

Geocode an address using the street centerline.

Service Arguments

PARAMETER DESCRIPTION
housenum House number.
prefix Address prefix. Example: 'N'.
streetname Street name.
roadtype Road type. Example: 'RD'.
suffix Street suffix.
zipcode Address zipcode.
city Address city.
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
x_coordinate X coordinate.
y_coordinate Y coordinate.
ll_add Lower left house number of street segment.
ul_add Upper left house number of street segment.
lr_add Lower right house number of street segment.
ur_add Upper right house number of street segment.
prefix Street prefix.
road_type Road type.
suffix Street suffix.
city City.
zipcode Zipcode.
