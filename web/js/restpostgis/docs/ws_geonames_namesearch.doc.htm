Geonames Name Search

Service URL

Description

Performs a fuzzy name search an entry in the geonames layer, which comes from GeoNames.org. Useful for auto complete boxes.

Service Arguments

PARAMETER DESCRIPTION
name Geoname to search for. Minimum of 2 characters.
format Format for the output. Value is either 'xml' for xml output, 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
geonameid Geonames ID.
name Geoname.
