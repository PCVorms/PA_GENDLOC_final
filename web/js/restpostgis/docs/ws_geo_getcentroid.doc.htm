Get Centroid

Service URL

Description

Returns the centroid(s) of the geometry of feature(s) from a spatial layer.

Service Arguments

PARAMETER DESCRIPTION
geotable The table containing the feature(s) to calculate a centroid from.
parameters SQL where parameters.
srid SRID of the centroid to be returned.
forceonsurface Value is either 'true' or 'false' (all lower case). A value of 'true' will force the returned centroid x and y to be inside of the subject geometry. A value of false will return the absolute centroid, which may be outside of the subject geometry.
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
x X coordinate of the centroid.
y Y coordinate of the centroid.
