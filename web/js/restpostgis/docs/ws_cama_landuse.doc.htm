CAMA Land Use

Service URL

Description

Returns CAMA land use information for a parcel ID.

Service Arguments

PARAMETER DESCRIPTION
pid Parcel ID.
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
parcel_id Parcel ID.
land_use Land use description of the parcel.
units Number of units.
neighborhood Neighborhood designation.
land_unit_type Land unit type code.
