CAMA Building

Service URL

Description

Returns CAMA building information for a parcel ID.

Service Arguments

PARAMETER DESCRIPTION
pid Parcel ID.
pidtype Type of parcel ID being passed. Value is either 'tax' or 'common'.
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
parcel_id Parcel ID.
common_parcel_id Common parcel ID.
card_number Tax card number.
property_use_description Property use description.
units Number of units.
year_built Year structure was built.
total_square_feet Total square feet of structure.
heated_square_feet Heated square feet of structure.
foundation_description Description of the structure's foundation.
exterior_wall_description Description of the exterior wall type.
heat_type Type of heating in structure.
ac_type Type of AC in structure.
stories Number of stories in structure.
bedrooms Number of bedrooms in structure.
full_baths Number of full baths.
three_quarter_baths Number of 3/4 baths.
half_baths Number of 1/2 baths.
building_type Type of building.
building_value Building value in dollars.
