Ubersearch

Service URL

Description

An autocomplete and search service involving a large amount of data and auto-quing between the search types. Output is in JSON. See Geoportal for an example of how this can work on a web site.

Service Arguments

PARAMETER DESCRIPTION
query The query string. Must be at least 3 characters. Searches for intersections are detected by an ampersand. Example: query=ruth & dolphin. Note the ampersand in the query string will have to be URL encoded.
searchtypes A comma delimited list of data sets to search. Example: searchtype=0,1,2,5. Here's the idex:
  • 0: Address (mat)
  • 1: Library (mat)
  • 2: School (mat)
  • 3: Park (mat)
  • 4: GeoName (mat)
  • 5: Street name (mat)
  • 6: CATS Light Rail (mat)
  • 7: CATS Park and Ride (mat)
  • 8: Intersection (mat)

Service Output

PARAMETER DESCRIPTION
query The query string sent to the service.
suggestions Array of suggestions. Format is [match-type]:[match].
data Data associated with each suggestion. Format is [search type]:[unique id field]:[unique id].
