CAMA PID Switcher

Service URL

Description

Returns both parcel ID types, tax and common, for a given parcel ID.

Service Arguments

PARAMETER DESCRIPTION
pid Eight character parcel ID. Example: 12312312.
pidtype Type of pid being input, either 'tax' for a tax parcel ID or 'common' for a common parcel ID, such as a condo common parcel. Both parcel types for the input parcel are returned.
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
parcel_id Tax parcel ID.
common_parcel_id Tax parcel common ID, for things like condo common areas.

Sample Output

Sample XML Output

	<rows total-rows="1">
	<row>
	<column name="parcel_id">11111111</column>
	<column name="common_parcel_id">11111111</column>
	</row>
	</rows>

Sample JSON Output

	{"total_rows":"1","rows":[{"row":{"parcel_id":"11111111",
	"common_parcel_id":"11111111"}}]}
