CAMA Sales History

Service URL

Description

Returns sales history for a parcel ID.

Service Arguments

PARAMETER DESCRIPTION
pid Parcel ID.
pidtype Type of parcel ID being passed. Value is either 'tax' or 'common'.
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
parcel_id Parcel ID.
common_parcel_id Common parcel ID.
sale_date Sale date (m/d/yyyy).
sale_price Sale price in dollars.
deed_book Deed book.
legal_reference Legal reference of sale.
