CAMA Ownership

Service URL

Description

The CAMA Ownership web service returns detailed parcel ownership information.

Service Arguments

PARAMETER DESCRIPTION
pid Eight character parcel ID. Example: 12312312.
pidtype Type of pid. Value is either 'tax' or 'common'. 'common' refers to a common pid, such as for a condo common area. In most cases you'll probably use 'tax'.
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
parcel_id Tax parcel ID.
common_parcel_id Tax parcel common ID, for things like condo common areas.
last_name Owner last name.
first_name Owner first name.
owner_type The type of owner as specified in the CAMA data.
address_1 Owner address first line.
address_2 Owner address second line.
city Owner address city.
state Owner address state.
zipcode Owner address zipcode.
