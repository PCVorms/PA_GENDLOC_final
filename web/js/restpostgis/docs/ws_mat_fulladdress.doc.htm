Full Address Search

Service URL

Description

Performs a fuzzy search for the full address, including city and zip, in the master address table. Useful for auto complete boxes.

Service Arguments

PARAMETER DESCRIPTION
address Address to search for. Minimum of 3 characters.
format Format for the output. Value is either 'xml' for xml output, 'json' for JSON output, and 'text' for plain text output.

Service Output

PARAMETER DESCRIPTION
objectid MAT object id.
address MAT address.
