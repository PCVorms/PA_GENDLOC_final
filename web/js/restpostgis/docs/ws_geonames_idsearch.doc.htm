Geonames ID Search

Service URL

Description

Retrieves information from the geonames table based on ID. From GeoNames.org.

Service Arguments

PARAMETER DESCRIPTION
id Geoname ID to search for.
format Format for the output. Value is either 'xml' for xml output, 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
geonameid Geonames ID.
name Geoname.
latitude Latitude in decimal degrees.
longitude Longitude in decimal degrees.
elevation In meters, integer.
population Population for things like cities.
gtopo30 Average elevation of 30'x30' area in meters (integer).
modification_date Date of the last modification.
