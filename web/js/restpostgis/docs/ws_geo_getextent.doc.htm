Get Extent

Service URL

Description

Get the extent of features from a specified geotable.

Service Arguments

PARAMETER DESCRIPTION
geotable The name of the geotable to perform the query on.
srid The projection of the returned extent.
parameters SQL where parameters.
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
extent The extent of the feature(s). Format: BOX(xmin ymin,xmax ymax)

Sample Output

Sample XML Output

	<rows total-rows="1">
	<row>
	<column name="extent">BOX(1445460.375 491995.21875,1454555 502776.15625)</column>
	</row>
	</rows>

Sample JSON Output

	{"total_rows":"1","rows":[{
	"row":{"extent":"BOX(1445460.375 491995.21875,1454555 502776.15625)"}}]}
