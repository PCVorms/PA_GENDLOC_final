CAMA Legal Description

Service URL

Description

Returns CAMA legal information for a parcel ID.

Service Arguments

PARAMETER DESCRIPTION
pid Parcel ID.
pidtype Type of parcel ID being passed. Value is either 'tax' or 'common'.
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
parcel_id Parcel ID.
common_parcel_id Common parcel ID.
legal_description Parcel legal description.
account_type Account type.
municipality Municipality.
fire_district Fire district.
total_acres Total acres.
