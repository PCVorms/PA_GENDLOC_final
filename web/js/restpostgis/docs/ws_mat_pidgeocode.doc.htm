PID Geocode

Service URL

Description

Return the master address table record(s) for a parcel ID.

Service Arguments

PARAMETER DESCRIPTION
pid Parcel ID.
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
objectid MAT address number.
x_coordinate X coordinate.
y_coordinate Y coordinate.
longitude Longitude.
latitude Latitude.
house_number House number.
prefix Street prefix. Example: 'N'
street_name Street name.
postal_city Postal city.
road_type The type of road. Example: 'RD'
suffix Street suffix.
unit House unit.
city Actual jurisdiction.
zipcode Zipcode.
parcel_id Tax parcel ID.
