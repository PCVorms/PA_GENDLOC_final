CAMA Appraisal

Service URL

Description

Returns CAMA appraisal information for a parcel ID.

Service Arguments

PARAMETER DESCRIPTION
pid Parcel ID.
pidtype Type of parcel ID being passed. Value is either 'tax' or 'common'.
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
parcel_id Parcel ID.
common_parcel_id Common parcel ID.
tax_year Tax year.
building_value Building value in dollars.
extra_features_value Extra features value in dollars.
land_value Land value in dollars.
total_value Total value in dollars.
