Stage Gage Readings

Service URL

Description

Retrieve the latest readings of all the Stage (Stream) Gages inside Mecklenburg County.

Service Arguments

PARAMETER DESCRIPTION
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
sensorid Stage Gage Number used by Charlotte Mecklenburg Storm Water Services (Alert).
usgs_station_no Stage Gage Number used by USGS (NWIS).
sitename Stage Gage Name.
longitude Longitude.
latitude Latitude.
streamlevel Last reported stream level in feet.
streamlevelsea Last reported stream level above mean sea level in feet.
