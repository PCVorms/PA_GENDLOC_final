Yahoo! Traffic

Service URL

Description

Gets a GeoRSS feed of traffic incident information from Yahoo!.

Service Arguments

PARAMETER DESCRIPTION
zip Zip code to use for traffic incidents. Leave as empty string if using longitude/latitude.
longitude Longitude to use for traffic incidents. Leave as empty string if using zip code.
latitude Latitude to use for traffic incidents. Leave as empty string if using zip code.
radius Radius in miles to use to search for traffic incidents around zip code or coordinate.

Service Output

Output format is that specified by Yahoo! and is XML-only. For more details, click here.
