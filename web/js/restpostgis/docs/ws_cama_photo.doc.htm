CAMA Photo

Service URL

Description

The CAMA Photo web service returns a URL to a house photo if one is available.

Service Arguments

PARAMETER DESCRIPTION
pid Eight character parcel ID. Example: 12312312.
pidtype Type of pid. Value is either 'tax' or 'common'. 'common' refers to a common pid, such as for a condo common area. In most cases you'll probably use 'tax'.
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
house_photo URL to the house photo if available.

Sample Output

Sample XML Output

	<rows total-rows="1">
	<row>
	<column name="house_photo">
	http://meckcama.co.mecklenburg.nc.us/mvideo/20000527/PR113423.JPG                  
	</column>
	</row>
	</rows>

Sample JSON Output

	{"total_rows":"1","rows":[{"row":
	{"house_photo":"http://meckcama.co.mecklenburg.nc.us/mvideo/20000527/PR113423.JPG"}}]}
