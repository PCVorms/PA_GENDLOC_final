Get Length and Area

Service URL

Description

Get the length and area of a string of points in x,y|x,y|x,y format. This is particularly useful with the Google Maps API v3 which, at this time, lacks the line.getLength() and polygon.getArea() in API v2

out_srid The projection for which you want to use for your length and area values. For example SRID 2264 outputs units in feet.

Service Arguments

PARAMETER DESCRIPTION
in_srid The projection of the string of input points.
out_srid The projection you want to use for your length and area values. For example SRID 2264 outputs units in feet.
points A string of points in x,y|x,y|x,y format for which you want to get length and area measurements.
length The length of the string of points in units of out_srid
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
length The length of the string of points in units of out_srid
area The area of the string of points in units of out_srid

Sample Output

Sample XML Output

	<rows total-rows="1">
	<row>
	<column name="length">1546.1343</column>
	<column name="area">76453.67</column>
	</row>
	</rows>

Sample JSON Output

	{"total_rows":"1","rows":[{
	"row":{"length":"1546.1343","area":"76453.67"}}]}
