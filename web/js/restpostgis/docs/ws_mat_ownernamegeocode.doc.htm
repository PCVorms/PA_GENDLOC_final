Owner Name Geocode

Service URL

Description

Returns master address table address for an owner name, joining the MAT and the CAMA ownership data via the parcel ID.

Service Arguments

PARAMETER DESCRIPTION
firstname Owner first name.
lastname Owner last name.
fuzzy Whether or not to perform a fuzzy search on the owner name. Must be 'true' or 'false'.
format Format for the output. Value is either 'xml' for xml output or 'json' for JSON output.

Service Output

PARAMETER DESCRIPTION
objectid MAT address number.
first_name Owner first name.
last_name Owner last name.
x_coordinate X coordinate.
y_coordinate Y coordinate.
longitude Longitude.
latitude Latitude.
house_number House number.
prefix Street prefix. Example: 'N'
street_name Street name.
postal_city Postal city.
road_type The type of road. Example: 'RD'
suffix Street suffix.
unit House unit.
city Actual jurisdiction.
zipcode Zipcode.
parcel_id Tax parcel ID.
